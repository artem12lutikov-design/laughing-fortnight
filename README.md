# 🎨 Мое первое портфолио

Это мой первый сайт, создаваемый для обучения веб-разработке!

## 📁 Структура проекта

```
laughing-fortnight/
├── index.html       # HTML-структура сайта
├── styles.css       # CSS-стили и дизайн
├── script.js        # JavaScript для интерактивности
└── README.md        # Этот файл
```

## 🎯 Что я изучаю

- ✅ **HTML** — структура веб-страниц
- ✅ **CSS** — стилизация и дизайн
- ✅ **JavaScript** — интерактивность и логика

## 🚀 Как просмотреть сайт

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/artem12lutikov-design/laughing-fortnight.git
   ```

2. Открой `index.html` в браузере

Или используй Live Server в VS Code для удобной разработки!
